# SDK 报错索引

这个文件用于回答一个问题：从报错或异常现象，反查应该看哪些 SDK 文档。

## 使用方式

- 用户贴出错误信息
- 用户描述“方法未定义”“类型转换错误”“找不到包”“调试起不来”
- 模型要从现象反查类、接口、类型或运行时边界

## 建设建议

- 每个条目包含：
  - 错误现象
  - 可能关联的 SDK 或类型
  - 第一排查项
  - 相关 FAQ
  - 相关示例

## 首批高频错误建议

- 依赖包无法识别
- 引擎未初始化
- 无法启动调试
- 长整型精度丢失
- 序列化问题
- 日期问题
- 接口 default 方法
- `QFilter` 类型转换错误
