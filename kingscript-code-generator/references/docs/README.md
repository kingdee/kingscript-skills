# KingScript 开发指南

本目录存放各开发类型的**专属**开发指南。公共注意事项、运行时约束和常见问题请见 `../gotchas/`。

## 当前文档

- `脚本控制器开发指南.md`
  - KWC（Kingdee Web Controller）脚本控制器开发指南
  - 包含控制器 XML 配置、URL 规则、权限配置、请求处理 API、响应处理 API

- `业务扩展点开发指南.md`
  - 业务扩展点开发指南
  - 包含扩展点基本结构、导出规范、约束引用

- `插件开发指南.md`
  - 插件开发指南
  - 包含 13 种插件类型速查、高频事件速查、选型决策树、起手建议

## 使用建议

- 用户提到 `KWC`、`脚本控制器`、`controller`、`REST API`、`Web API` 时，阅读 `脚本控制器开发指南.md`
- 用户提到 `扩展点`、`extension`、`extpoint` 时，阅读 `业务扩展点开发指南.md`
- 用户提到 `插件`、`plugin` 时，阅读 `插件开发指南.md`
- 涉及金额/日期/DynamicObject/BigInt 等**公共运行时约束**，请到 `../gotchas/`
- 如果还需要具体类、方法或声明，再回到 `../sdk/`
- 如果还需要示例代码，到 `../examples/`
