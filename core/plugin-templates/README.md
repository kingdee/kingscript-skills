# Kingscript Plugin Templates

这个目录存放可以直接复用或稍作替换后使用的 Kingscript 插件模版和脚手架。

## 建议存放

- 按钮事件模板
- 表单事件模板
- 数据校验模板
- 调后端服务模板
- 异常处理模板

## 每个模板建议包含

- 适用场景
- 依赖的上下文对象
- 占位参数
- 风险说明
- 与示例的关联

## 当前可直接参考的模板

- [bill-before-property-changed-template.md](bill-before-property-changed-template.md)
  - 适合单据场景下的字段修改前拦截
  - 适合做审核后禁止修改、负数校验、状态校验

## 使用建议

- 先确认插件类型和事件时机，再选模板
- 先替换字段标识、状态值、提示语，再放入业务代码
- 模板只能解决结构问题，运行时字段、权限和开放能力仍需按实际环境确认
