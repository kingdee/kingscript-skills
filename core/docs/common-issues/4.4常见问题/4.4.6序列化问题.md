### 问题描述
`KingScript` 在序列化时可能遇到序列化结果不完整或者无法反序列化等问题

### 问题分析
`KingScript` 目前提供了两种序列化能力
1. `Js` 原生的序列化方法：`JSON.stringify` 与 `JSON.parse`
- JSON.stringify
```kingscript
//json data
let jsonData = {
    "name":"jack",
    "age":20
};
JSON.stringify(jsonData);
//'{"name":"jack","age":20}'

//list data
let listData = [1,2,3];
JSON.stringify(listData);
//'[1,2,3]'
```
- JSON.parse
```kingscript
//json string
let jsonString = '{"name":"jack","age":20}'; 
JSON.parse(jsonString);
/*
{
    "name":"jack",
    "age":20
}
*/

//json list
let jsonList = '[1,2,3]'
JSON.parse(jsonList);
//[1,2,3]
```
`JSON` 序列化用于格式化 `js` 原生对象，如：json、数组对象等
2. 平台对象序列化工具类 `SerializationUtils` 提供的数据对象序列化与反序列化接口
- SerializationUtils.toJsonString
```kingscript
//DymamicObject object
let objType:MainEntityType = EntityMetadataCache.getDataEntityType("test_ksbill");
let obj:DynamicObject = new DynamicObject(objType);
obj.setValue("billno","number1");
let jsonStr = SerializationUtils.toJsonString(obj);
console.log(jsonStr);
/*
{
    "id":0,
    "billno":"number1",
    "billstatus":null,
    "creator":null,
    "modifier":null,
    "auditor":null,
    "auditdate":null,
    "modifytime":"2025-03-18 11:10:53",
    "createtime":"2025-03-18 11:10:53",
    "entryentity":[]
}
*/

//Map object
let mapObj:HashMap = new HashMap();
mapObj.put("key","value");
let jsonStr = SerializationUtils.toJsonString(mapObj);
console.log(jsonStr);
//{"key":"value"}
```
- SerializationUtils.fromJsonString
```kingscript
let jsonStr = '{"key":"value"}';
let mapObj:HashMap = SerializationUtils.fromJsonString(jsonStr, HashMap);
let v = mapObj.get("key");
console.log(v);
//value
```
`SerializationUtils` 工具类用于格式化 `java` 对象，如：List、HashMap、DynamicObject等

> ##### 根本原因
> 在 `KingScript` 中出现序列化与反序列化异常通常是 `js` 原生对象与 `java` 对象混用所致，如使用 `SerializationUtils` 序列化 `json`、`[]` 等原生 `js` 对象

### 解决办法
将 `js` 与 `java` 对象分开处理，如果需要传递值时，先使用对应序列化工具处理完再传递，如：
```kingscript
let mapObj:HashMap = new HashMap();
let jsonObj = {
    "name":"jack"
};
mapObj.put("key",JSON.stringify(jsonObj));
let jsonStr = SerializationUtils.toJsonString(mapObj);
console.log(jsonStr);
//{"key":"{\"name\":\"jack\"}"}
```

```kingscript
let jsonStr = '{"key":"{\"name\":\"jack\"}"}';
let mapObj:HashMap = SerializationUtils.fromJsonString(jsonStr, HashMap);
let v = mapObj.get("key");
let jsonObj = JSON.parse(v);
console.log(jsonObj);
//{"name":"jack"}
```
