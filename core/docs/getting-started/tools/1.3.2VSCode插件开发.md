# 一、前言
## 1.插件介绍
金蝶云苍穹 VSCode 开发助手插件是一款助力 `KingScript` 轻脚本开发者进行应用开发的工具，支持创建本地工程、上传、下载脚本，本地 debug 调试，还兼备许多实用功能。

# 七、实用功能
## 1.脚本模板
为了方便本地开发，本插件内置了常用的脚本插件模板，新建文件后默认使用表单插件模板。其他模板可通过以下方式来使用：

方式一 关键字补全
输入kd:，找到要生成的模板，使用Tab或Enter补全即可生成。

![img_26.png](images/img_26.png)

方式二 命令框选择
Ctrl + Shift + P打开命令选择框，输入 `insert snippet` ，选择需要的模板，点击即可生成。

![img_27.png](images/img_27.png)

## 2.脚本片段

方式一 关键字补全

输入kd:sample，找到要生成的片段，使用Tab或Enter补全即可生成。

![img_28.png](images/img_28.png)

支持预览

![img_29.png](images/img_29.png)

可按需调节窗口大小

![img_30.png](images/img_30.png)

使用 ↑ ↓ 方向键可以快速切换片段进行预览

![img_31.png](images/img_31.png)

方式二 命令框选择

Ctrl + Shift + P打开命令选择框，输入 `insert snippet` ，选择需要的片段，点击即可生成。
