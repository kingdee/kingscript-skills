# SDK 插件与扩展点索引

这个文件用于回答一个问题：某个插件类型或扩展点应该去哪里找对应 SDK。

## 使用方式

- 用户问“某种插件能用哪些接口”
- 用户知道插件场景，不知道对应扩展点名称
- 用户需要从插件类型反查可用 SDK

## 推荐组织方式

- 按插件类型分组
  - 表单
  - 单据
  - 列表
  - 报表
  - 调度
  - 工作流
  - 打印
  - 引入引出
- 每组列出：
  - 常用基类
  - 高频事件
  - 高频扩展点接口
  - 相关示例
  - 常见坑
